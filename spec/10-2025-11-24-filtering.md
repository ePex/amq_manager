# Feature Request: Filtering Lists

**Date:** 2025-11-24
**Status:** Planned

## Description
The user needs to filter lists to find specific items quickly.
- **Queue List**: Filter by Queue Name.
- **Message List**: Filter by `JMSType`.

## Requirements
- **Interaction**:
  - Press `/` to open a filter input bar (like `vim` or `less`).
  - Typing in the input filters the list in real-time or upon pressing Enter.
  - Pressing `Esc` clears the filter/closes the input.
- **Queue List**:
  - Filter logic: Case-insensitive substring match on Queue Name.
- **Message List**:
  - Filter logic: Case-insensitive substring match on `JMSType` property.

## UI/UX
- **Filter Bar**: A `Input` widget docked at the bottom (above Footer) or top.
- **Visibility**: Hidden by default, shown when `/` is pressed.
- **Focus**: Automatically focused when shown.
