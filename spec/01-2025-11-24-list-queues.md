# Feature Request: List Queues

**Date:** 2025-11-24
**Status:** Implemented

## Description
The user needs to see a list of all available queues on the ActiveMQ broker to understand the current state of the system.

## Requirements
- Connect to the ActiveMQ broker via Jolokia API.
- Retrieve a list of all queues.
- Display key statistics for each queue:
  - **Queue Name**
  - **Pending Messages** (QueueSize)
  - **Consumer Count**
  - **Enqueued Count**
  - **Dequeued Count**
- Provide a mechanism to refresh the list to see the latest data.

## UI/UX
- A table view in the terminal.
- Columns for each statistic.
- Rows selectable to proceed to message viewing.
